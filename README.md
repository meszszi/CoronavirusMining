# Coronavirus Mining

## Installation
```bash
$ pip install -r requirements.txt
```

## Usage
```bash
$ cd application/
$ python visualize.py
```

###Flags
 - ``--country alpha_3`` - parametrize script with alpha_3 country code available [here](https://www.iban.com/country-codes), default POL
 
###Examples
 ```bash
$ cd application/
$ python visualize.py --country USA
```
